총평

기존 보고서는 .env 노출, JWT, XSS, 쿠키 등 인프라/설정 레벨의 취약점은 잘 포착했으나, 실제 코드 레벨에서의 인증 미적용
API(30개+), 테스트 코드 잔존(인증 완전 우회), NoSQL Injection을 통한 권한 상승, Socket.IO 접근 제어, OAuth 구현
취약점 등이 누락되어 있었습니다.

특히 가장 심각한 3가지 누락:
1. test-code로 소셜 로그인 완전 우회 — 지금 즉시 제거 필요
2. 인증 없이 계정 재활성화(탈취)/아카이브(삭제) — 지금 즉시 인증 추가 필요
3. 채팅 메시지 무인증 열람 — 통신비밀보호 위반 소지

한국 법률 측면에서는 14세 미만 아동 보호, 유출 사고 72시간 신고 의무, 전자상거래 기록 5년 보관, AWS 국외이전 동의,
징벌적 과징금 10% 개정 등이 누락되어 있어 법적 리스크가 과소평가되어 있었습니다.
